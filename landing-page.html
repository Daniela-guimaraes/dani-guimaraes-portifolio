Visão Geral e Desafio Técnico

Desenvolvimento de uma Landing Page de Alta Conversão focada na venda de um produto artesanal. O desafio foi criar uma experiência de compra dinâmica, utilizando lógica de programação para simular um fluxo de carrinho simples e aplicar técnicas de persuasão em vendas (gatilhos mentais).

O foco técnico principal foi a manipulação do DOM via JavaScript para criar uma experiência de usuário fluida e reter o estado da compra.

Habilidades e Aprendizados Destacados:

  • Implementação de cálculo dinâmico em JavaScript para ajustar preço total com base na quantidade selecionada.
  • Uso de localStorage para garantir a persistência da quantidade selecionada, melhorando a UX (User Experience).
  • Configuração de gatilhos mentais (Contagem Regressiva, Escassez e Âncora de Preço) puramente via HTML, CSS e JavaScript.
  • Validação de *inputs* de usuário (quantidade mínima/máxima) diretamente no front-end.
  • Controle e customização de elementos de formulário (`input type="number"`) via CSS Webkit para superar bugs de compatibilidade.

Design Focado em Conversão e Escassez

Implementação do `localStorage` para salvar a última quantidade selecionada pelo cliente. Isso garante que, ao voltar para a página, o estado anterior seja restaurado, eliminando a frustração do reset e otimizando o fluxo de compra (ótimo ponto de UX!).

Checkout Simplificado e Integrado ao PIX

Tela final de checkout com foco na clareza e facilidade de pagamento. Implementação de fluxo de pagamento via PIX (QR Code e Copia e Cola), seguido de confirmação via WhatsApp para automatizar o processo e garantir a reserva do produto.

Captura de Leads com Cupom Exclusivo

Seção de pop-up ou modal para captura de e-mails (leads), oferecendo um desconto atrativo para a primeira compra. Essencial para construir lista de clientes e remarketing. Foco: UX/UI simples e direto.

Manipulação da URL e Cálculo de Total (Front-end)

Função JavaScript para ler dados da URL (total e qtd) utilizando URLSearchParams. O script carrega e formata o valor total do pedido, preparando o dado para ser exibido e configurando o link de confirmação do pedido no WhatsApp.
